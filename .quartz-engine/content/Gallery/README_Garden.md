This folder is your **Staging Area** for the Digital Garden.

## How to Publish
1.  **Draft** your content anywhere in `4 Hevel`.
2.  **Polish** it for the public.
3.  **Move/Copy** the final file into **this folder** (`4 Hevel/Garden`).
4.  **Deploy**: Run the `Deploy-Garden` script (or ask Bird/Jim).

Any file placed here will automatically appear in the **Gallery** section of `garden.hevel.ca`.

*Note: Subfolders here will be preserved in the Gallery.*
